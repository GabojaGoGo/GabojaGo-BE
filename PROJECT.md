# TripMate

> 2026 관광데이터 활용 공모전 | Flutter + Spring Boot

---

## 인증
- 카카오 OAuth PKCE 로그인 (딥링크 콜백)
- JWT 액세스/리프레시 토큰 자동 갱신
- 게스트 모드 지원

## 홈
- GPS 기반 주변 관광지 (10km, 최대 100개)
- 주변 축제 목록
- 관광지별 혼잡도 (DB 스냅샷 → 예측 → API 순 폴백)

## 주변 관광지 지도
- 카카오맵 마커 + 카드 연동
- 지도 이동 시 "이 지역 검색" 재조회
- 진입 시 100개 전체 로드

## 관광지 상세
- 한국관광공사 API 실시간 연동
- 기본정보 / 운영정보 / 이용요금 / 이미지 갤러리

## 여행 취향 설정
- 목적(최대 3개) + 기간 선택
- 코스 추천 / 재설정 분리

## 기록 탭
- 족적: 방문 지역 그리드 (DB 연동)
- 혜택 리포트: 누적 절약 금액 (DB 연동)
- 버킷리스트: 추가 / 완료 / 삭제 (DB 연동)
- 취향 프로필 표시
- 여행자 레벨 뱃지 (족적 수 기준, 탭 시 기준 표시)

## 혜택/보조금
- 카테고리별 보조금 목록
- 지역별 예산 잔여율 + 신청 안내

## 플래너
- 여행 코스 추천 화면

## 백엔드
- Flyway DB 마이그레이션
- 개인정보 AES-256 암호화
- 혼잡도 일별 스냅샷 스케줄러
- 한국관광공사 API 프록시 (반경/개수 파라미터)
