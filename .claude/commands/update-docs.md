# CLAUDE.md / 커맨드 파일 갱신

이번 세션에서 변경된 내용을 기준으로 아래 항목을 갱신하세요.

## 체크리스트

1. **CLAUDE.md 갱신 대상**
   - 새로 추가된 파일/화면 → `## 디렉토리` 섹션
   - 새로 추가된 API 엔드포인트 → `## API 엔드포인트 요약` 섹션
   - 새로 발견된 주의사항·패턴 → `## 핵심 규칙` 또는 `## 자주 쓰는 패턴`
   - 변경된 설정값 (반경, 개수 등) → 해당 섹션

2. **커맨드 파일 갱신 대상** (`.claude/commands/`)
   - `db-integrate.md` — DB 연동 패턴이 바뀐 경우
   - `fix-overflow.md` — 새 오버플로우 유형 발견 시
   - `add-endpoint.md` — 백엔드 컨벤션 변경 시
   - `check-auth.md` — 새 인증 이슈 패턴 발견 시

## 규칙
- 삭제된 내용은 지우고, 추가된 내용만 간결하게 추가
- 변경 없으면 파일 건드리지 말 것
- 코드 예시는 실제 프로젝트 코드 기준으로 작성
